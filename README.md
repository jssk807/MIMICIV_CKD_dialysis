# MIMICIV_CKD_dialysis
The following project will analyze the the efficacy of the 2024 KDIGO guideline-recommended pharmacological treatments—specifically, ACEi, ARBs, SGLT2i, and statins—in delaying the time to dialysis among patients with CKD by utilizing the MIMIC IV dataset.

# Data Availability
The dataset used for this study is the MIMIC-IV (Medical Information Mart for Intensive Care IV) database. As MIMIC-IV contains sensitive Protected Health Information (PHI), access is strictly controlled and governed by a Data Use Agreement (DUA).
*The raw data files are NOT included in this repository.*

## How to Access MIMIC-IV
To obtain and download the MIMIC-IV dataset, researchers must first complete the following steps:
1.  **Complete Training:** Pass the **Citi Program course "Data or Specimens Only Research"** (Course ID: 17192).
2.  **Request Access:** Register an account on the **PhysioNet** platform and submit a request to access the MIMIC-IV project.
3.  **Review Documentation:** Familiarize yourself with the official **MIMIC Code Repository** and documentation on PhysioNet.

**Official PhysioNet MIMIC-IV Project Page:**
https://physionet.org/content/mimiciv/3.1/
